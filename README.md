# VRMEditorTools

ブラウザ上でVRMモデルの確認、テクスチャ変更、表情の撮影などができるツールです。

## 主な機能

### 1. モデル確認・編集
- **VRM読み込み**: VRMファイルをドラッグ&ドロップまたは選択して読み込み。
- **マテリアル・テクスチャ管理**: マテリアルごとのテクスチャ確認や差し替えが可能。
- **3面図保存**: 正面、横、背面からの画像をボタン一つで保存。

### 2. ブレンドシェイプ (BlendShape)
- **表情一覧**: 定義されているブレンドシェイプ（表情）をボタンで切り替えて確認できます。
- **Face Focus**: ブレンドシェイプタブを選択すると、自動的にキャラクターの顔にフォーカスします。
- **一括ダウンロード (Download All)**: 全ての表情を自動で撮影し、ZIPファイルとしてまとめてダウンロードできます。

### 3. その他
- **画像一覧**: モデルに含まれる画像リソースの確認。
- **視点・スケール調整**: カメラ位置やモデルサイズの調整。
- **VRoid向け機能**: VRoid Hub連携や最適化機能（詳細はお問い合わせください）。

## 環境・開発

- Node.js v24.12.0
- Vue.js 2.x
- Firebase Hosting

### セットアップ

```bash
# Node.js v24.12.0 の利用を推奨
nodenv install 24.12.0

# 依存関係のインストール
npm i
```

### ビルド

```bash
npm run build
```

### 開発サーバーの起動

```bash
npm run serve
```
`http://localhost:8080` で起動します。
